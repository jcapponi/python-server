# python-server

Ejemplo de servidor HTTP en Python con Flask.

## Descripción

Este proyecto corresponde al backend del ejemplo y expone una API REST para:

- crear textos
- leer textos
- actualizar textos
- borrar textos

## Ubicación

Crear el directorio:

`/Python-Server/backend`

Copiar allí el archivo `app.py`.

## Ejecución

1. Abrir la carpeta `backend`
2. Ejecutar el servidor con:

   ```bash
   python app.py
3. Comprobar que esté corriendo en:

http://127.0.0.1:5000
